# Laravel HRMS Demo

This is a demo HR Management System built using Laravel.

## Features
Developed a comprehensive HR Management System using PHP Laravel following MVC architecture.
The system includes Admin and Employee authentication with role-based access control.

Key modules implemented:
- User Authentication (Admin / Employee)
- Employee Management
- Attendance Management
- Leave Management
- Payroll & Salary Slip Generation
- Notice Board, Warning, Training & Promotion modules
- Opportunities, Leads & Accounts
- Product & Vendor Management
- Purchase Order & Invoice Management
- Calendar & Award Management
- Role & Permission Management
- REST APIs for module communication
- MySQL Database design with proper relationships

Focused on backend development, database structure, validations, security and clean code practices.
This is a demo project created for portfolio purposes without using company proprietary code.


## Tech Stack
- PHP
- Laravel
- MySQL
- HTML, CSS, Bootstrap
- JavaScript, AJAX
